# Java Internship Projects

A collection of Java console applications developed during my internship to practice core programming concepts and problem solving.

## Projects

### Student Management System
Manage student records with add, view, search, update, and delete operations.

### Scientific Calculator
Perform basic arithmetic and scientific calculations including trigonometric and logarithmic functions.

### Electricity Billing System
Calculate electricity bills based on units consumed and predefined tariff rates.

### School Result Management
Generate student results with total marks, percentage, grades, and pass/fail status.
